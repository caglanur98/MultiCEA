# MultiCEA

**MultiCEA** is a research repository for studying **causal evidence accumulation in multisensory perception**.  
The project focuses on how the brain integrates, segregates, and dynamically evaluates multisensory signals under uncertainty, combining **behavioral experiments**, **computational modeling**, and **neuroimaging approaches** such as **fMRI** and **EEG**.

A central experimental approach in this project is the use of an **audiovisual temporal rate task** to investigate both:

- **explicit causal judgments** — directly asking whether audiovisual signals are perceived as originating from a common cause or from separate causes
- **implicit causal inference** — examining how causal structure influences perceptual or decision responses even when causality is not explicitly reported

The repository is designed to support a broader neurocognitive framework including:

- **Behavioral task scripts**
- **Bayesian Causal Inference (BCI) modeling**
- **CI-DDM / evidence accumulation models**
- **fMRI analysis pipelines**
- **EEG analysis pipelines**
